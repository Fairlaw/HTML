				<p>A Linguagem de Folhas de Estilo em Cascata, conhecida como CSS (Cascading Style Sheets),
				 é uma das tecnologias fundamentais da World Wide Web (WWW). Foi desenvolvida em conjunto com o
				  HTML para permitir a estilização e o design de páginas da web de forma mais eficiente e consistente.
				   O CSS foi proposto pela primeira vez por Håkon Wium Lie e Bert Bos em 1994 e, desde então,
				    tem sido essencial para tornar a experiência de navegação mais agradável e visualmente atraente.
					</p> <h3>Propósito do CSS:</h3>
				<p>O propósito principal do CSS é separar a apresentação visual do conteúdo
				 estrutural em uma página da web. Enquanto o HTML é responsável por definir a estrutura e os elementos da
				  página, o CSS é responsável por controlar a aparência desses elementos. Isso significa que os
				   desenvolvedores podem criar um único documento HTML e aplicar diferentes estilos a ele usando folhas de estilo em cascata.
				   </p>

				<p>A Linguagem JavaScript é uma das principais tecnologias da web moderna. Criada por Brendan Eich em 1995,
				 ela se tornou uma das linguagens de programação mais populares e amplamente utilizadas. 
				 Originalmente desenvolvida para ser executada nos navegadores web, o JavaScript evoluiu para ser utilizado em 
				 uma variedade de ambientes, incluindo servidores, aplicativos móveis e até Internet das Coisas (IoT).</p>
				<h3>Propósito do JavaScript:</h3>
				<p>O propósito principal do JavaScript é adicionar interatividade e dinamismo às páginas da web.
				 Enquanto o HTML é responsável pela estrutura e o conteúdo da página, e o CSS pelo estilo e a 
				 aparência, o JavaScript é a linguagem que permite criar ações e comportamentos no lado do cliente. 
				 Com o JavaScript, é possível manipular elementos da página, responder a eventos do usuário e criar 
				 animações e transições suaves.</p>
