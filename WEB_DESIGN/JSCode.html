Este website é apenas uma avaliação sobre UC de Desenvolvimento Web
                    
                        
                        O código abaixo é um script em JavaScript que cria um acordeão (accordion) interativo para uma 
                        lista de elementos com a classe "accordion". Quando um elemento "accordion" é clicado, ele expande 
                        ou recolhe o painel seguinte, permitindo que o conteúdo seja exibido ou ocultado.</br>

                        O script começa obtendo todos os elementos com a classe "accordion" usando o método getElementsByClassName().
                         Em seguida, ele itera por cada um desses elementos usando um loop for. Para cada elemento, é adicionado um
                          ouvinte de eventos (addEventListener) que reage quando o elemento é clicado.
                        
                        Dentro do manipulador de eventos, o elemento atual que foi clicado é referenciado como "this". É então alternada 
                        a classe "active" no elemento atual, o que pode ser útil para aplicar estilos ao elemento ativo.
                        
                        Em seguida, é obtido o próximo elemento irmão (nextElementSibling) do elemento atual, que deve ser o painel 
                        associado ao acordeão. O painel é armazenado na variável "panel".
                        
                        Depois, verifica-se se o estilo "maxHeight" do painel está definido. Se estiver definido (ou seja, o painel já
                         está expandido), é removido (tornando-o recolhido). Caso contrário, é definido o "maxHeight" do painel como a
                          altura total do conteúdo (scrollHeight), fazendo com que o painel seja expandido para mostrar todo o conteúdo.
                        
                        Em resumo, este script cria um acordeão funcional que expande e recolhe o conteúdo conforme o usuário clica nos
                         títulos dos painéis.
                        

                        <script>
                        var acc = document.getElementsByClassName("accordion");
                        var i;
                        
                        for (i = 0; i < acc.length; i++) {
                        acc[i].addEventListener("click", function() {
                            this.classList.toggle("active");
                            var panel = this.nextElementSibling;
                            if (panel.style.maxHeight) {
                            panel.style.maxHeight = null;
                            } else {
                            panel.style.maxHeight = panel.scrollHeight + "px";
                            }
                        });
                        }
                        </script>
